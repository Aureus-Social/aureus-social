# Exemple Activa — Nordin MOUSSATI

Données basées sur l’**attestation Activa.brussels** (Actiris) pour Nordin MOUSSATI.

## Option 1 : Bouton dans l’app (recommandé)

1. Menu **GESTION RH & PAIE** → **Liste Employes**.
2. Cliquer sur **« Exemple Activa (Nordin MOUSSATI) »**.
3. L’employé est créé et vous êtes redirigé vers **Fiches de Paie**.
4. Choisir **Nordin MOUSSATI**, puis dans **Activation ONEM** : **Activa.brussels AP (350→800→350)**.
5. Lancer le calcul : l’allocation (350 € / 800 € / 350 € selon le mois) est calculée automatiquement.

## Option 2 : Import CSV

1. **Liste Employes** → **Import Excel**.
2. Sélectionner le fichier :  
   `docs/import-exemple-activa-nordin-moussati.csv`
3. Après import, aller dans **Fiches de Paie** → sélectionner **Nordin MOUSSATI** → **Activation ONEM** : **Activa.brussels AP (350→800→350)**.
